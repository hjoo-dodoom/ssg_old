  • 04작성한 주문파일

  • 05주문 확인 후
    [오류체크] 클릭
  • 06오류 수정 후
    [주문신청] 클릭
※ 주문 신청 시 안내사항
  1. 1. 주문 파일 등록 시 엑셀파일에 작성한 내용 중 잘못 작성한 경우 화면에서 수정할 수 있습니다
  2. 2. 반드시 [주문신청]버튼을 클릭 하셔야 주문이 완료됩니다
  3. 3. 승인이 완료된 건은 [발송준비중]으로 표시되며, 우측 하단의 [주문/배송 조회]를 선택하시면 확인하실 수 있습니다
  4. 4. 상품 발송 시, 송장표기는 “주문자” 그대로 발송됩니다 [예) 홍길동(○), ○○점 홍길동(Ⅹ)]
주문 가이드 다운로드이용가이드ㆍ주문샘플파일
※ 주문 파일 등록 유의사항
  1. 1. DRM 문서보안이 적용된 파일이나 암호화 파일인 경우 보안 해지 또는 암호화 해지 후 업로드 해주시기 바랍니다
  2. 2. 다른 파일에서 내용을 가져올 경우, 서식/함수 등은 반드시 제거하여야 합니다

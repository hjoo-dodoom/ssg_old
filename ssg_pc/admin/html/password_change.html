  • 3. 공백이 포함된 경우

  • 1. 공백없이 8~12자리의 영문(대/소문자), 숫자, 특수문자 조합
  • 2. 연속된 숫자, 개인정보가 포함된 비밀번호의 경우 노출위험이 있으니 사용을 삼가해 주세요.
